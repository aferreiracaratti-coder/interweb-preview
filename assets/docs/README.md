# Interweb Salto – Asset pack web

Assets listos para armar una landing con scroll basada en el mockup:

## Imágenes principales
- `images/hero-interweb-desktop.png` — hero desktop 1920x1080.
- `images/hero-interweb-mobile.png` — hero vertical mobile 1080x1600.
- `images/repair-devices-cutout.png` — teléfonos/tablet recortados con fondo transparente.
- `images/cta-repair-background.png` — fondo oscuro para CTA de WhatsApp.
- `images/about-section-interweb.png` — bloque visual “Sobre nosotros”.
- `images/storefront-interweb-salto.png` — foto del local recortada y escalada.
- `images/security-camera-installation.png` — imagen para cámaras de seguridad.
- `images/map-placeholder-salto.png` — placeholder visual para contacto/mapa.
- `images/service-*.png` — tarjetas visuales por servicio.

## Logos
- `logos/interweb-logo-primary.svg`
- `logos/interweb-logo-white.svg`
- `logos/interweb-mark.svg`

## Iconos SVG
Incluye iconos en azul y blanco: celular, batería, carga, cámara, agua, software, garantía, WhatsApp, ubicación, horario, PC, CCTV y herramientas.

## UI
- `ui/interweb-tokens.css` — variables de color, radio y sombra.
- `ui/brand-logos-strip.svg` — tira de marcas como recurso visual.
- `ui/favicon.svg`

Colores base: azul técnico, cyan, blanco, gris claro y rojo de acento para la submarca.
